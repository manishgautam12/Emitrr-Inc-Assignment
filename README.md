## Tech Stack

### Backend

<li>Express</li>

</ul>

### Frontend
<ul>
<li>React.js</li>
<li>CSS</li>
<li> Meterial UI </li>
</ul>

## Getting Started


### Backend


Run the following commands to start the backend server
```sh 
cd backend
npm install
node index.js
   or
nodemon index.js
```


### Frontend

Run the following commands to start the React App
```sh
cd client
npm install
npm start
```
